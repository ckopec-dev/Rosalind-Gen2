# Rosalind Problem: Genome Assembly Using Reads (Ada Solution)

## Problem Understanding

The genome assembly problem involves reconstructing a complete DNA sequence from overlapping short sequences (reads) that were generated by sequencing technology. We need to find the shortest superstring that contains all given reads as substrings.

## Approach

I'll use a greedy approach where:
1. Find the maximum overlap between any two reads
2. Merge the reads with maximum overlap
3. Repeat until only one read remains

## Solution in Ada

```ada
with Ada.Text_IO; use Ada.Text_IO;
with Ada.Strings.Unbounded; use Ada.Strings.Unbounded;
with Ada.Integer_Text_IO; use Ada.Integer_Text_IO;

procedure Genome_Assembly_Using_Reads is
   
   type Read is record
      Sequence : Unbounded_String;
      Length   : Natural;
   end record;
   
   type Read_Array is array (Positive range <>) of Read;
   
   function Overlap_Length(Left, Right : Unbounded_String) return Natural is
      -- Calculate maximum overlap between left and right strings
      Max_Overlap : Natural := 0;
      L_Length    : Natural := Length(Left);
      R_Length    : Natural := Length(Right);
      Min_Length  : Natural := Natural'Min(L_Length, R_Length);
   begin
      -- Check all possible overlaps from left to right
      for I in 1..Min_Length loop
         if To_String(Left)(L_Length - I + 1 .. L_Length) = 
            To_String(Right)(1 .. I) then
            Max_Overlap := I;
         end if;
      end loop;
      
      return Max_Overlap;
   end Overlap_Length;
   
   function Find_Max_Overlap(Reads : Read_Array) return (Natural, Natural) is
      -- Find the pair of reads with maximum overlap
      Max_Overlap : Natural := 0;
      Best_Left   : Natural := 0;
      Best_Right  : Natural := 0;
      L_Length    : Natural := Reads'Length;
   begin
      for I in Reads'First..Reads'Last loop
         for J in Reads'First..Reads'Last loop
            if I /= J then
               declare
                  Overlap : Natural := Overlap_Length(Reads(I).Sequence, 
                                                     Reads(J).Sequence);
               begin
                  if Overlap > Max_Overlap then
                     Max_Overlap := Overlap;
                     Best_Left   := I;
                     Best_Right  := J;
                  end if;
               end;
            end if;
         end loop;
      end loop;
      
      return (Best_Left, Best_Right);
   end Find_Max_Overlap;
   
   function Merge_Reads(Left, Right : Unbounded_String; Overlap : Natural) 
      return Unbounded_String is
      -- Merge two reads with given overlap
      L_Length : Natural := Length(Left);
      R_Length : Natural := Length(Right);
      Result   : Unbounded_String;
   begin
      if Overlap = 0 then
         -- No overlap, just concatenate
         Result := Left & Right;
      else
         -- Remove overlapping part from right and concatenate
         Result := Left & 
                   To_Unbounded_String(To_String(Right)(Overlap + 1 .. R_Length));
      end if;
      
      return Result;
   end Merge_Reads;
   
   procedure Print_Reads(Reads : Read_Array) is
   begin
      for I in Reads'First..Reads'Last loop
         Put_Line("Read " & Integer'image(I) & ": " & To_String(Reads(I).Sequence));
      end loop;
   end Print_Reads;

begin
   -- Example test case (replace with actual input reading)
   -- For demonstration, let's use a simple example:
   
   declare
      Reads : Read_Array(1..4) := (
         (To_Unbounded_String("AACCTTGG"), 8),
         (To_Unbounded_String("TTGGCCAA"), 8),
         (To_Unbounded_String("CCAAATTT"), 8),
         (To_Unbounded_String("TTTTCCAA"), 8)
      );
      
      L : Natural := Reads'Length;
   begin
      -- Print initial reads
      Put_Line("Initial reads:");
      Print_Reads(Reads);
      New_Line;
      
      -- Greedy assembly process
      while L > 1 loop
         declare
            Left_Index, Right_Index : Natural;
            Overlap_Value           : Natural;
         begin
            -- Find maximum overlap
            Left_Index := Find_Max_Overlap(Reads).First;
            Right_Index := Find_Max_Overlap(Reads).Second;
            
            if Left_Index /= 0 and Right_Index /= 0 then
               -- Calculate actual overlap
               Overlap_Value := Overlap_Length(Reads(Left_Index).Sequence, 
                                              Reads(Right_Index).Sequence);
               
               -- Merge the reads
               declare
                  New_Read : Unbounded_String := 
                     Merge_Reads(Reads(Left_Index).Sequence,
                                Reads(Right_Index).Sequence,
                                Overlap_Value);
               begin
                  -- Replace the left read with merged result
                  Reads(Left_Index).Sequence := New_Read;
                  Reads(Left_Index).Length   := Length(New_Read);
                  
                  -- Remove the right read by shifting elements
                  for I in Right_Index..L-1 loop
                     Reads(I) := Reads(I+1);
                  end loop;
                  
                  L := L - 1;
               end;
            else
               exit; -- No more overlaps found
            end if;
         end;
      end loop;
      
      Put_Line("Assembled sequence:");
      Put_Line(To_String(Reads(1).Sequence));
   end;
   
end Genome_Assembly_Using_Reads;
```

## Key Features of the Solution

1. **Data Structures**: 
   - `Read` record to store DNA sequences and their lengths
   - `Read_Array` for handling multiple reads

2. **Overlap Calculation**:
   - `Overlap_Length` function computes maximum overlap between two strings
   - Compares suffix of left string with prefix of right string

3. **Greedy Assembly**:
   - `Find_Max_Overlap` identifies the best pair to merge
   - `Merge_Reads` combines reads with calculated overlap
   - Repeatedly merges until one final sequence remains

4. **Memory Management**:
   - Uses Ada's Unbounded_String for dynamic string handling
   - Efficient array manipulation during merging process

## Time Complexity

- Overlap calculation: O(n × m) where n is string length, m is number of reads
- Overall complexity: O(n² × m²) in worst case

## Usage Notes

To use this with actual Rosalind input:

This implementation provides a solid foundation for genome assembly using the greedy approach, which works well for many practical sequencing scenarios.
