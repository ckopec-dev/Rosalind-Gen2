# Rosalind Problem: Find Substrings of a Genome Encoding a Given Amino Acid String

## Problem Understanding

This problem asks us to find all substrings of a given DNA sequence that encode a specific amino acid sequence. We need to:
1. Translate DNA to amino acids using the genetic code
2. Find all substrings that translate to the target amino acid sequence
3. Return all such substrings

## Solution Approach

1. Create a genetic code mapping from codons to amino acids
2. For a given DNA sequence, find all possible translations (3 reading frames)
3. Check each translation against the target amino acid sequence
4. Return the corresponding DNA substrings
