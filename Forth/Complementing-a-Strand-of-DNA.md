# Rosalind Problem: Complementing a Strand of DNA in Forth

## Problem Understanding

The task is to find the reverse complement of a DNA strand. Given a DNA string consisting of nucleotides A, C, G, T, we need to:
1. Complement each nucleotide (A↔T, C↔G)
2. Reverse the resulting string
