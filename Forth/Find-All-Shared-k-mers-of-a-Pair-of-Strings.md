# Rosalind Problem: Find All Shared k-mers of a Pair of Strings (Forth Solution)

## Problem Understanding

We need to find all shared k-mers (k-length substrings) between two DNA strings and return their positions in both strings.

## Solution Approach

1. Generate all k-mers from both strings
2. Find common k-mers between both sets
3. For each common k-mer, find all positions in both strings
4. Return positions in the required format
