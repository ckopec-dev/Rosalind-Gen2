# Rosalind Problem: Find the Longest Repeat in a String (Forth Solution)

## Problem Understanding

The task is to find the longest substring that appears at least twice in a given string, where the occurrences can overlap.

## Solution Approach

We'll use a brute-force approach in Forth:
1. Generate all possible substrings
2. Check if each substring appears at least twice
3. Keep track of the longest valid substring
