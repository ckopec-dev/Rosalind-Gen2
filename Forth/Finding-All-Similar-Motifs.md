# Rosalind Problem: Finding All Similar Motifs (Forth Solution)

## Problem Understanding

The problem asks us to find all motifs (short DNA sequences) that appear in all given DNA sequences. This is essentially finding the intersection of all possible motifs of a given length across multiple sequences.

## Approach

1. Parse input DNA sequences
2. Generate all possible motifs of a given length from each sequence
3. Find motifs that appear in ALL sequences
4. Return the common motifs
