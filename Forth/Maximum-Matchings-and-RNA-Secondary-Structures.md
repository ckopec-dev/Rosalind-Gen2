# Rosalind Problem: Maximum Matchings and RNA Secondary Structures in Forth

## Problem Understanding

The problem asks us to find the maximum number of base pairings in an RNA sequence such that:
1. Each nucleotide can pair with its complement (A↔U, C↔G)
2. Base pairs cannot cross each other (no pseudoknots)
3. We want to maximize the number of valid base pairs

This is a classic dynamic programming problem that can be solved using the Nussinov algorithm.

## Solution Approach

We'll use dynamic programming where `dp[i][j]` represents the maximum number of base pairs in the subsequence from position i to j.
