# Rosalind Problem: Overlap Alignment in Forth

## Problem Understanding

Overlap alignment is a variant of sequence alignment where we want to find the best alignment between two sequences that allows free gaps at the beginning and end of the sequences, but penalizes internal gaps. The goal is to maximize the alignment score.

## Solution Approach

I'll implement the overlap alignment algorithm using dynamic programming with a scoring matrix, then trace back to find the optimal alignment.

## Forth Implementation

```forth
\ Overlap Alignment in Forth
\ Based on the Smith-Waterman algorithm with free gaps at ends

\ Constants
: MATCH ( -- n ) 2 ;
: MISMATCH ( -- n ) -1 ;
: GAP ( -- n ) -1 ;

\ Matrix operations
: matrix-init ( rows cols -- )
    create
    dup * cells allocate throw
    does> ( row col -- addr )
        over cells * swap cells + + ;

\ Global matrix for dynamic programming
100 100 matrix dp-matrix

\ Overlap Alignment Function
: overlap-align ( seq1 seq2 -- score )
    \ Get sequence lengths
    dup 0 ?do 1+ loop  \ Length of seq2
    swap 0 ?do 1+ loop  \ Length of seq1
    1+ 1+  \ Add 1 for zero index
    >r >r
    
    \ Initialize DP matrix
    0 0 0 dp-matrix !
    
    \ Fill first row (free gaps at beginning)
    1 r@ ?do
        i 0 0 dp-matrix !
    loop
    
    \ Fill first column (free gaps at beginning)
    1 r> ?do
        0 i 0 dp-matrix !
    loop
    
    \ Fill the rest of the matrix
    1 r> ?do
        1 r@ ?do
            \ Calculate scores for three possibilities
            i 1- j 1- dp-matrix @  \ Match/mismatch
            i 1- j dp-matrix @  \ Deletion (gap in seq1)
            i j 1- dp-matrix @  \ Insertion (gap in seq2)
            
            \ Calculate match score
            i 1- j 1- dp-matrix @
            i 1- j dp-matrix @
            i j 1- dp-matrix @
            
            \ Find maximum
            3dup max max max
            
            \ Store in matrix
            i j 0 dp-matrix !
        loop
    loop
    
    \ Return maximum score (last cell)
    r> 1- r> 1- dp-matrix @
;
