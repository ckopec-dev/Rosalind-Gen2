# Rosalind Problem: Creating a Distance Matrix (LMC Solution)

## Problem Understanding

The task is to create a distance matrix from a set of DNA sequences. The distance between two sequences is typically measured as the proportion of nucleotide positions at which the sequences differ (p-distance).

## LMC Solution

```lmc
; Creating a Distance Matrix
; Rosalind problem solution in LMC

; Data section
DATASET
    ; Example sequences (in practice, these would be read from input)
    ; Using 4 sequences of length 5
    "ACGTG"
    "ACGTT"
    "ACGAG"
    "ACGTC"
    ; Number of sequences
    4
    ; Length of each sequence
    5

; Main program
START   LDA COUNT
        STA I
        LDA LENGTH
        STA J
        LDA NUMSEQ
        STA N
        LDA 0
        STA DISTANCE
