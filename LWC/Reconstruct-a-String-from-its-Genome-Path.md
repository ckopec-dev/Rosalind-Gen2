# Rosalind Problem: Reconstruct a String from its Genome Path

## Problem Statement
Reconstruct a string from its genome path (a sequence of k-mers).

## Solution in Lightning Web Component

```javascript
// reconstructStringFromGenomePath.js
import { LightningElement } from 'lwc';

export default class ReconstructStringFromGenomePath extends LightningElement {
    genomePath = '';
    result = '';

    handleInputChange(event) {
        this.genomePath = event.target.value;
    }

    handleReconstruct() {
        if (!this.genomePath.trim()) {
            this.result = 'Please enter a genome path';
            return;
        }

        const path = this.genomePath.trim().split('\n').filter(line => line.trim());
        if (path.length === 0) {
            this.result = 'Invalid input';
            return;
        }

        this.result = this.reconstructString(path);
    }

    reconstructString(genomePath) {
        if (genomePath.length === 0) {
            return '';
        }

        // First k-mer contributes all its characters
        let result = genomePath[0];
        
        // For each subsequent k-mer, add only the last character
        for (let i = 1; i < genomePath.length; i++) {
            result += genomePath[i][genomePath[i].length - 1];
        }
        
        return result;
    }

    handleClear() {
        this.genomePath = '';
        this.result = '';
    }
}
```

## How It Works

1. **Input**: Takes a genome path as input, where each line represents a k-mer
2. **Algorithm**: 
   - The first k-mer contributes all its characters to the result
   - For each subsequent k-mer, only the last character is added to the result
3. **Output**: Returns the reconstructed string
