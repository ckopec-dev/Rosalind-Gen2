# Rosalind Problem: Convert a Peptide into a Peptide Vector

## Problem Statement
Given a peptide, we need to convert it into its peptide vector representation. The peptide vector is a binary vector of length 3n (where n is the length of the peptide) where each position represents a possible amino acid (represented as integers 0-20) and the vector indicates whether that amino acid appears at that position in the peptide.

## Solution Approach
1. Create a mapping from amino acids to integers (0-20)
2. For each amino acid in the peptide, convert it to its integer representation
3. Build the peptide vector by placing 1 at positions corresponding to amino acid indices

## Implementation

```python
def peptide_to_vector(peptide):
    """
    Convert a peptide into its peptide vector representation.
    
    Args:
        peptide (str): A string representing the peptide sequence
    
    Returns:
        list: Binary vector representing the peptide
    """
    # Amino acid to integer mapping (0-20)
    acid_to_int = {
        'A': 0, 'C': 1, 'D': 2, 'E': 3, 'F': 4, 'G': 5, 'H': 6,
        'I': 7, 'K': 8, 'L': 9, 'M': 10, 'N': 11, 'P': 12, 'Q': 13,
        'R': 14, 'S': 15, 'T': 16, 'V': 17, 'W': 18, 'Y': 19
    }
    
    # Length of peptide vector is 3 * length of peptide
    vector_length = 3 * len(peptide)
    peptide_vector = [0] * vector_length
    
    # Fill the vector
    for i, acid in enumerate(peptide):
        # Each amino acid contributes to 3 positions in the vector
        position = i * 3 + acid_to_int[acid]
        peptide_vector[position] = 1
    
    return peptide_vector

# Read input from file or stdin
def main():
    # Read peptide from input
    peptide = input().strip()
    
    # Convert to peptide vector
    vector = peptide_to_vector(peptide)
    
    # Print result
    print(' '.join(map(str, vector)))

if __name__ == "__main__":
    main()
```
