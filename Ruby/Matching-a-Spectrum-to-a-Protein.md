# Rosalind Problem: Matching a Spectrum to a Protein

## Problem Description
Given a spectrum and a protein string, determine if the protein's theoretical spectrum matches the given spectrum.

## Solution

```ruby
def read_spectrum(filename)
  File.readlines(filename).map(&:chomp).map(&:to_f)
end

def amino_acid_masses
  {
    'A' => 71.03711,
    'C' => 103.00919,
    'D' => 115.02694,
    'E' => 129.04259,
    'F' => 147.06841,
    'G' => 57.02137,
    'H' => 137.05891,
    'I' => 113.08406,
    'K' => 128.09496,
    'L' => 113.08406,
    'M' => 131.04049,
    'N' => 114.04293,
    'P' => 97.05276,
    'Q' => 128.05858,
    'R' => 156.10111,
    'S' => 87.03203,
    'T' => 101.04768,
    'V' => 99.06841,
    'W' => 186.07931,
    'Y' => 163.06333
  }
end

def protein_spectrum(protein)
  masses = amino_acid_masses
  spectrum = [0.0]  # Start with 0 (theoretical spectrum starts with 0)
  
  protein.chars.each do |aa|
    mass = masses[aa]
    spectrum << mass
  end
  
  # Generate all prefix sums for the spectrum
  prefix_sums = [0.0]
  spectrum.each_with_index do |mass, i|
    if i > 0
      prefix_sums << prefix_sums.last + mass
    end
  end
  
  # Remove the first element (0) and return the spectrum
  prefix_sums[1..-1]
end

def spectra_equal?(spectrum1, spectrum2, tolerance = 0.001)
  return false if spectrum1.length != spectrum2.length
  
  spectrum1.zip(spectrum2).all? do |mass1, mass2|
    (mass1 - mass2).abs <= tolerance
  end
end

def match_spectrum_to_protein(spectrum_file, protein)
  # Read the spectrum from file
  spectrum = read_spectrum(spectrum_file)
  
  # Generate theoretical spectrum for the protein
  theoretical_spectrum = protein_spectrum(protein)
  
  # Compare spectra
  spectra_equal?(spectrum, theoretical_spectrum)
end

# Example usage:
# spectrum_file = "rosalind_spec.txt"
# protein = "SKADYEK"
# result = match_spectrum_to_protein(spectrum_file, protein)
# puts result

# Alternative implementation for the specific problem format
def solve_matching_a_spectrum_to_a_protein
  # Read input
  input = File.readlines("rosalind_spec.txt").map(&:chomp)
  spectrum = input[0..-2].map(&:to_f)  # All but last line are spectrum values
  protein = input[-1]  # Last line is protein string
  
  # Generate theoretical spectrum
  theoretical = protein_spectrum(protein)
  
  # Compare
  if spectra_equal?(spectrum, theoretical)
    puts "true"
  else
    puts "false"
  end
end

# For the actual Rosalind problem, we might need:
def solve_with_protein_and_spectrum(protein, spectrum)
  theoretical_spectrum = protein_spectrum(protein)
  spectra_equal?(spectrum, theoretical_spectrum)
end
```

## Explanation

This solution addresses the problem of matching a theoretical protein spectrum to an experimental spectrum:

1. **Amino Acid Masses**: The `amino_acid_masses` hash contains the monoisotopic masses for all 20 amino acids.

2. **Protein Spectrum Generation**: The `protein_spectrum` function:
   - Takes a protein string as input
   - Calculates the theoretical spectrum by summing amino acid masses
   - Returns prefix sums representing the spectrum (including 0 at the start)

3. **Spectrum Comparison**: The `spectra_equal?` function:
   - Compares two spectra element by element
   - Uses a tolerance of 0.001 to account for floating-point precision
   - Returns true if all differences are within tolerance

4. **Main Logic**: The solution reads the spectrum from file and compares it with the theoretical spectrum of the given protein.

## Usage Example

```ruby
# If you have a file with spectrum values and a protein string
spectrum = [0.0, 113.08406, 226.16812, 339.25218, 452.33624]
protein = "SKADYEK"

result = match_spectrum_to_protein(spectrum, protein)
puts result  # Will output true or false
```

The key insight is that a protein spectrum is generated by taking prefix sums of amino acid masses, starting from 0, and then comparing this theoretical spectrum with the experimental spectrum.
